## 🗂️ 디렉토리

- index : `App` 함수와 html을 연결해주는 파일
- assets : 페이지 구성에 필요한 자료 저장하는 폴더
- constants : 상수화된 파일들이 저장된 폴더
- data : 시각화에 필요한 자료들을 저장한 폴더
- js : 로직을 실행하는 js 파일 자료를 저장하는 폴더
- utils : 파일의 가독성을 위해 분리한 함수를 저장하는 폴더
